SUDOKU

Rule 0: Each region of the sudoku is equals a 9x9 squares more small

Rule 1: Type in each empty box a number between 1 - 9

Rule 2: In the same row can not be repeated the same number

Rule 3: In the same columm can not be repeated the same number

Rule 4: Cannot be repeated the same number in the same region

Rule 5: The solve is unique
